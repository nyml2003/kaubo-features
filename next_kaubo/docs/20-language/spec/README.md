## 快速概览

### 示例程序

```kaubo
// 结构体定义
struct Point {
    x: float,
    y: float
}

// 方法实现
impl Point {
    distance: |self, other: Point| -> float {
        var dx = self.x - other.x;
        var dy = self.y - other.y;
        return std.sqrt(dx * dx + dy * dy);
    }
}

// 使用
var p1 = Point { x: 0.0, y: 0.0 };
var p2 = Point { x: 3.0, y: 4.0 };
print(p1.distance(p2));  // 5.0
```

### 核心特性

- **静态类型**：编译时类型检查，无隐式转换
- **热重载**：修改代码1秒内生效，不丢失状态
- **运算符重载**：通过 `operator add` 等方式自定义运算
- **协程**：`yield` 支持生成器模式
- **模块**：`module`/`import`/`from` 组织代码

### 基础类型

| 类型 | 说明 | 示例 |

| `Function` | 函数/闭包 | `\|x\| x + 1` |

## 与实现的关系

```
语言规范 (spec/)
    ↓ 约束
实现细节 (impl/)
    ↓ 具体化
源代码 (src/)
```

- **规范变化** = 语法/语义改变（影响用户代码）
- **实现变化** = 架构优化/bug修复（不影响用户代码）
